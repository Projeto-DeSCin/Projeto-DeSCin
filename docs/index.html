DeSCin

🧬 DeSCin Blockchain Node

Um nó de blockchain focado em Ciência Descentralizada (DeSci), construído em C++17 moderno. Este projeto implementa a estrutura base de blocos, criptografia de carteiras (Wallets), e testes unitários automatizados, utilizando CMake como sistema de build.

💻 Acessar Plataforma 👨🏽‍💻 Acessar Código 📄 Ler Relatório ▶️ Vídeo no YouTube

✨ O que o nosso sistema faz

  • Estrutura de Blocos Segura: Criação, mineração e encadeamento de blocos utilizando hashes criptográficos (SHA-256).
  • Gestão de Carteiras (Wallets): Geração de chaves assimétricas (públicas/privadas) utilizando algoritmo RSA para segurança das transações.
  • Rede e API: Integração com a biblioteca de rede Asio e rotas via Crow API para comunicação eficiente entre nós.
  • Testes Automatizados: Cobertura de testes unitários com GoogleTest para garantir a integridade das funções criptográficas.

💻 Ambientes Suportados

Este projeto foi desenhado para rodar em ambientes Linux.

  • Ubuntu (Nativo): Suporte total e nativo.
  • Windows 10/11: Suporte total via WSL 2.

🛠️ Pré-requisitos

🪟 Usuários de Windows (Preparando o WSL)

  1. Abra o PowerShell como Administrador e rode: wsl --install
  2. Reinicie o computador.
  3. Instale o VS Code e a extensão WSL.

🚀 Instalação e Compilação

1. Clonar o Repositório


        git clone https://github.com/GabrielCassio/Projeto-DeSCin.git
        cd Projeto-DeSCin

2. Instalar Dependências


        sudo apt update
        sudo apt install build-essential cmake git libasio-dev -y

3. Compilar o Projeto


        cmake -B build
        cmake --build build

🎯 Executando

Nó Principal:


        ./build/descin_blockchain

Testes Unitários:


        ./build/descin_tests
